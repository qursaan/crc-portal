In this experiment, a remote user is allowed to upload a binary file into specific IoT device. The operations held behind is that:

  1. User is authenticated ensured to have valid slice on specific resource
  2. the file is uploaded to the servers local storage via HTTP protocol
  3. The device configuration machine is contacted to issue an AVRDUDE command with specific parameters such as com port and baud-rate.

    4. The board from this moment acts as indicated in the binary file.
