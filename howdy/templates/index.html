Welcome to Assiut IoT Testbed

The main goal of Assiut IoT Testbed is to aid students/researchers to create and monitor experiments related to IoT via a remotely configurable IoT test bed that is consists of different types of cores (Arduino, Atmega16 uc, STM ARM, Raspberry PI and NodeMCU) with different communication capabilities (LTE, 3G, WIFI, Zigbee....). Also, each node will be equipped by the commonly used sensors (Temperature, humidity, ...etc) The nodes are remotely accessible, after central reservation, via the, in-site, server that is connected to the different nodes core. User (researcher or student) will be able to reserve some of the available nodes for a certain time then access the cores of the reserved nodes (Arduino for example) and load his firmware to build his desired topology.
